# CreditWise Loan Approval System

## 📌 Overview
This project is a Machine Learning-based loan approval system that predicts whether a loan application should be *Approved or Rejected* based on applicant details such as income, credit score, employment status, etc.

The goal is to automate the loan approval process and reduce manual errors, bias, and processing time.

---

## 📊 Dataset Features
- Applicant Income
- Coapplicant Income
- Employment Status
- Age
- Marital Status
- Dependents
- Credit Score
- Existing Loans
- Loan Amount
- Savings, Collateral Value, etc.

---

## ⚙️ Project Workflow
1. *Data Preprocessing*
   - Handled missing values using SimpleImputer
   - Encoded categorical variables (Label Encoding)
   - Feature scaling using StandardScaler

2. *Exploratory Data Analysis (EDA)*
   - Visualized data using Matplotlib & Seaborn
   - Analyzed distribution of Loan Approval, Credit Score, etc.

3. *Feature Engineering*
   - Created new features (e.g., log transformation, squared features)

4. *Model Building*
   - Logistic Regression  
   - K-Nearest Neighbors (KNN)  
   - Naive Bayes  

5. *Model Evaluation*
   - Accuracy Score  
   - Precision  
   - Recall  
   - F1 Score  
   - Confusion Matrix  

---

## 🏆 Results
- *Naive Bayes achieved the highest precision* among all models.
- Compared performance of Logistic Regression and KNN models.
- Selected best model based on evaluation metrics.
